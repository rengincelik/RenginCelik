## Featured Project

### [PipeFlow Simulator](https://github.com/rengincelik/pipeFlow)

A browser-based, real-time pipeline flow simulator built for process engineers.

### [Live Demo →](https://rengincelik.github.io/pipeFlow/)

**Technologies:** Vanilla JavaScript (ES Modules), SVG Rendering, Canvas 2D  
**Dependencies:** Zero external dependencies

**Key Features:**
- Real-time simulation updated every 100ms
- Drag & drop pipeline construction (pipes, pumps, valves, elbows, reducers, etc.)
- Live visualization of pressure, velocity, flow rate, and energy losses
- Interactive SVG animations and dynamic charts

**Physics Implemented:**
- Darcy-Weisbach equation with Colebrook-White friction factor (solved via Newton-Raphson)
- Borda-Carnot and contraction losses with Bernoulli pressure recovery
- Pump H-Q curve fitting and operating point calculation using Bisection method
- Temperature-dependent fluid properties (density & viscosity)
